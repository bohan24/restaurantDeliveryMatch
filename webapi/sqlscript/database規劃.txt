資料庫規劃: 

資料庫使用 postgreSQL免安裝載點 : https://github.com/pgsql-tw/portable-pgsql 
可以使用免安裝版的方式跑開發資料庫

資料表開頭大寫 欄位駝峰

會員表(會員分級 可停用)
Member
  Id uuid,varchar(50),PrimaryKEY,
  name,varchar(50),null,
  email,varchar(200),null,
  account,varchar(500),
  password,varchar(4000),
  isAuth,bool,1登入，0登出,default(0),
  lineUid,varchar(200),null,
  isUse,bool,1使用，2停用,default(1),
  createDate,datetime,
  deleteDate,datetime,null
  upadteDate,datetime,null

廠商表 v
Client
  Id uuid,varchar(50),Primarykey,
  name,varchar(200),null
  number,varchar(8),備註:統一編號,null
  address,varchar(500),null
  category,int,備註:服務類型,
  isUse,bool,1使用，2停用,default(1),
  createDate,datetime,
  deleteDate,datetime,null
  upadteDate,datetime,null
  
廠商管理員表 
Admin
  Id uuid,varchar(50),PrimaryKEY,
  clientId,varchar(50),
  name,varchar(50),null,
  email,varchar(200),null,
  account,varchar(500),
  password,varchar(4000),
  isAuth,bool,1登入，0登出,default(0),
  lineUid,varchar(200),null,
  payment,text, 備註:陣列
  isUse,bool,1使用，2停用,default(1),
  createDate,datetime,
  deleteDate,datetime,null
  upadteDate,datetime,null
 
權限類型 
Premission
  Id uuid,varchar(50),PrimaryKEY,  
  name,varchar(50),null,
  isUse,bool,1使用，2停用,default(1)
  
訂單表 (可消取功能)
Order
    Id uuid,varchar(50),PrimaryKEY,
    clientId,varchar(50),
    productId,text, 備註:陣列
    memberId,varchar(50),
    deliveryId,varchar(50),
    price,text, 備註:陣列
    total,int, 備註:總金額
    paymentId,varchar(50), 備註:付款方式
    address,varchar(255), 取貨地址
    pickerTime,datetime,
    createDate,datetime,
    updatedDate,datetime,null,
    deletedDate,datetime,null
    
支付選項
Payment
    Id uuid,varchar(50),PrimaryKEY,
    name,varchar(50),null,
    isUse,bool,1使用，2停用,default(1)
    
產品表 (通常每個系統產品表不一樣)
Product
    Id uuid,varchar(50),PrimaryKEY,
    name,varchar(50),null,
    adminId,varchar(50),
    Introduction,text, 備註:產品說明,
    category,varchar(50),
    price,int, 備註:商品單價
    createDate,datetime,
    updatedDate,datetime,null,
    deletedDate,datetime,null

系統設定表(排程時間預定 系統規格 之類設定)
SystemConfig
